# Руководство пользователя (Русский)

## Пользовательский сценарий

1. Экспортировать чат Telegram в JSON.
2. Загрузить JSON в Chat Analyzer.
3. Дождаться завершения анализа.
4. Изучить дашборд, графики и таблицы.

## Как экспортировать Telegram (`result.json`)

1. Откройте **Telegram Desktop**.
2. Откройте конкретный чат, который хотите анализировать.
3. Нажмите меню с тремя точками в правом верхнем углу.
4. Выберите **Export chat history**.
5. Укажите формат **JSON** и включите историю сообщений.
6. Отключите медиа, чтобы уменьшить размер файла.
7. Запустите экспорт, дождитесь завершения и найдите JSON-файл (часто `result.json`).

## Загрузка и анализ

1. Откройте Web UI (`http://localhost:8080`).
2. Загрузите экспортированный Telegram JSON.
3. Дождитесь статуса `done`.
4. Откройте дашборд и проверьте метрики/графики/таблицы.

## Примечания по данным

- ПО self-hosted: данные остаются в вашем окружении.
- Загруженные файлы и сгенерированные JSON-результаты хранятся в backend-хранилище.
