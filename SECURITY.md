# Security Policy

## Reporting a Vulnerability

Please do **not** open public GitHub issues for security vulnerabilities.

Send a private report to: `alievsaid0909@yandex.ru`

Please include:

- affected component and version,
- reproduction steps or proof of concept,
- impact assessment,
- suggested mitigation (if available).

## Response SLA

- Initial response target: within 72 hours.
- Status updates are shared during triage and remediation.

## Disclosure

After fix deployment, responsible disclosure and coordinated public notes are encouraged.
