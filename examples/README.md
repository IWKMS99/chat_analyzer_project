# Example Dataset

`example_chat.json` is a synthetic Telegram-like export for smoke testing.

## Data origin

- Fully synthetic content created for this repository.
- No real personal chats or personally identifiable information (PII).

## Limitations

- Smaller than real production exports.
- Not representative of every Telegram export edge case.
- Intended for quick onboarding and UI/API sanity checks.

## How to use

1. Start the app (`docker compose` or local dev stack).
2. Open the web UI.
3. Upload `examples/example_chat.json`.
4. Wait for status `done` and open dashboard.
